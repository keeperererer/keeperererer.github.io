创建vue-cli项目

  • npm install vue-cli -g
  • vue init webpack projectName

把本地项目与github相连

  • 在github新建一个仓库,复制git链接
  • 在本地 git clone 链接
  • cd 到项目位置
  • git init
  • git commit -m ‘…’
  • git remote add origin git@github(git地址)
  • git pull origin master
  • git push origin master
